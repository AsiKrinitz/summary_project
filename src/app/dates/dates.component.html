Dates Explain in Angular

in order to use Date in angular we can use a object called Date

we can get the current date by using a simple method -
todayDate : Date = new Date();
we just create a property type of Date and creating a new object of it and it will get our current time in the country -

{{ todayDate }}



if you want you can change the format of the date with pipes !

short pipe

A shortDate format that displays the date and time in the locale-specific short format. For example, "M/d/yy h:mm a" in the en-US locale.

example to short pipe - {{ todayDate | date : "short" }}


medium pipe

A medium format that displays the date and time in the locale-specific medium format. For example, "MMM d, y h:mm:ss a" in the en-US locale.

exapmle to medium pipe - {{ todayDate | date : "medium" }}


long pipe

A long format that displays the date and time in the locale-specific long format. For example, "MMMM d, y h:mm:ss a z" in the en-US locale.

exapmle to medium pipe - {{ todayDate | date : "long" }}
